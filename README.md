### Instalação:

     ruby '3.0.2'
     gem rails '7.0.4'
     
Em sua maquina acesse a pasta do projeto e execute:

      bundle install
      
Apos:
     rails db:create
     rails db:migrate
     
Para subir o projeto local execute:

     rails s
     
Por fim, em seu browser entre no endereço mecionado no terminal
